## Conclusions for reconstruction

1. **Use v1.10 as the baseline** for source reconstruction — it represents the
   most complete rewrite from v1.02 and v1.11 is a small patch on top.
2. **v1.02 → v1.10** requires essentially a full re-decompilation (different
   function addresses throughout).
3. **v1.10 → v1.11** can be represented as a patch set: ~96 small diffs,
   mostly branch-target fixups from one or two function insertions.
4. **Data/coefficient tables** (0x059B00+) are **identical** across all three
   versions — extract once, share across all builds.
5. **String tables** (0x050000–0x059A54) differ mainly in the version banner
   — share the table template, override version strings per build.
