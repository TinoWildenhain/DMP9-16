# Security policy

This repository contains reverse engineering research on vintage commercial
hardware firmware (Yamaha DMP9/16, manufactured 1991-1994).

## Scope

This project does not distribute ROM images or copyrighted firmware.
See `roms/README.md` for the ROM acquisition policy.

## Responsible disclosure

If you identify a security issue in the tooling in this repository
(emulator harness, Ghidra scripts, build system), please open a
GitHub issue tagged `security`.

## ROM images

Do not open issues or pull requests attaching ROM binary files.
ROM images are the intellectual property of Yamaha Corporation.
